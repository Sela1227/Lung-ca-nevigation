# 彰濱秀傳癌症中心 — 癌症臨床路徑導航系統

多癌別臨床路徑管理系統。單一入口，每個癌別獨立目錄與版本。

## 目錄結構

```
index.html          ← 入口頁 (portal v1)
lung/index.html     ← 肺癌 v27
lung/edu.html       ← 肺癌 QR 掃碼頁
breast/             ← 乳癌（建構中）
crc/                ← 大腸直腸癌（建構中）
esoph/              ← 食道癌（建構中）
hn/                 ← 頭頸癌（建構中）
liver/              ← 肝癌（建構中）
prost/              ← 攝護腺癌（建構中）
bladder/            ← 膀胱癌（建構中）
```

**部署網址：** `https://sela1227.github.io/Lung-ca-nevigation/`

## 版本規則

- **入口頁 (portal)**：portal v1, v2…（改入口頁時升）
- **各癌別**：獨立版號（如 lung v27），改該癌別時升
