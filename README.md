- **系統版號 (SYSTEM_VERSION)**：整個 Cancer Navigation 的 release 編號

## 系統版本歷史

### V2.4.0 — 2026-04-06
**肺癌檢查清單與治療選項調整**
- 移除基礎檢查的「病史及理學檢查」（一定會做，不需勾選）
- 輔助治療「合併化放療」更名為「術後化放療」（僅在 adjuvant 函式內，不影響一線 CCRT）
- 系統性藥物 → 免疫子組新增「免疫 + 化療 + 標靶」三合一選項
- KPI 策略檢查同步加入新選項與 `target_chemo`
- **模組變動**：lung V1.1.2 → V1.2.0

### V2.3.3 — 2026-04-06
**確立通用打包標準（wrapper 結構 + 排除清單）**

### V2.3.2 — 2026-04-06
**修正分子檢測無法直接補填的不直覺操作**
- **模組變動**：lung V1.1.1 → V1.1.2

### V2.3.1 — 2026-04-06
**修正列印衛教手冊版型**
- **模組變動**：lung V1.1 → V1.1.1

### V2.3.0 — 2026-04-06
**Nordic 自製 SVG 圖示系統**
- **模組變動**：lung V1.0 → V1.1

### V2.2.0 — 2026-04-06
**雙軌版本系統導入**
- **模組變動**：lung V2.1.1 → V1.0（首個穩定版）

### V2.1.1 — 2026-04-06
**打包結構修正** — 後來被 V2.3.3 取代

### V2.1.0 — 2026-04-06
**視覺大改：肺癌頁面統一 Nordic misty blue + 移除所有 emoji**

### V2.0.0 — 2026-04-06
**專案重命名為 Cancer Navigation + 標準化**
